# 🎉 Project Information

This project is designed for temporary party collaboration in SpigotMC.

### ✨ Features

- 🧟 Shared EXP when killing monsters  
- 💰 Money sharing via external plugins  
- ❤️ Display players' health and hunger in the Tab menu
